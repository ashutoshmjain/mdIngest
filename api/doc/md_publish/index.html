§md-publish (The Ingestion Layer)

A professional, modular asset ingestion bridge for mdbook. This crate serves as the Ingestion Layer within an autonomous Research-to-Publish Workflow.

§Agentic Architecture

md-publish is designed to operate as a standalone module that can be triggered by a Research Agent. It handles:

  1. Sanitization: Transforming raw AI output into hardened Markdown.
  2. Media Management: Migrating and renaming assets based on a Master Key (Episode Number).
  3. Indexing: Maintaining the book’s structure and social/monetization widgets.

Currently, the tool includes specialized logic for Google Gemini Pro (Shielded Output stripping), with future support planned for other LLMs like ChatGPT and Claude.
